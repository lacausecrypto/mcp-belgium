# Domain Catalog

## Live no-key domains

- `irail`: Belgian rail journeys, stations, liveboards, disturbances
- `mobility`: intermodal public transport planning
- `infrabel`: rail infrastructure datasets and records
- `best-address`: official address and municipality data
- `statbel`: population, CPI, employment, dataset search
- `air-quality`: stations, measurements, BelAQI, timeseries
- `brussels`: Brussels Region open datasets and records
- `wallonia`: Wallonia open datasets and records
- `kmi`: KMI/IRM weather GeoServer layers and observations
- `geoservices-wallonia`: ArcGIS REST discovery and layer queries
- `urban-brussels`: WFS feature types and GeoJSON retrieval
- `urbis`: public WMS layer discovery and GetMap URL building

## Key-gated domains

- `stib`: requires `STIB_API_KEY`
- `kbo`: requires `CBEAPI_KEY`

## Compatibility layers

- `data-gov-be`: older federal CKAN contract preserved, but upstream contract changed
- `flanders`: older CKAN-like contract preserved, but current upstream requires an API key

## Discovery tools

- `belgium_catalog_overview`
- `belgium_list_domains`
- `belgium_describe_domain`
